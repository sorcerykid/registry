Player Registry Mod v1.0
By Leslie E. Krause

Player Registry provides an API for simple aggregation of player properties.

Repository
----------------------

Browse source code...
  https://bitbucket.org/sorcerykid/registry

Download archive...
  https://bitbucket.org/sorcerykid/registry/get/master.zip
  https://bitbucket.org/sorcerykid/registry/get/master.tar.gz

Compatability
----------------------

Minetest 0.4.14+ required

Installation
----------------------

  1) Unzip the archive into the mods directory of your game
  2) Rename the registry-master directory to "registry"
  3) Add "registry" as a dependency to any mods using the API
